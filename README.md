## MOTIVO
Este projeto tem como proposito a obtenção de nota da matéria Projeto Integrador II.
## IDEALIZADORES
Neste projeto estão os seguintes alunos: Maéli, Maikon, Marlon, Julio e Veneranda.
Estamos sendo orientados por: Ana Isabel da Fonseca.
## RESUMO
Ao termos de ouvir a comunidade externa, decidimos atender a demanda de uma empresa de nome Chiptronic, da qual o problema principal é a falta de um sistema computadorizado de fácil acesso para o agendamento da frota automotiva. Será elaborado um sistema web para que ajude que ela realize facilmente, consulte e adapte os agendamentos. A entrega e acessibilidade será de forma gradual, respeitando etapas e feedbacks.
